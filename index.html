01§23

개념 단어 목록 구성

대상 정의
논문

"happy", "sad", "calm", "desperate" 같은 감정 개념을 나타내는 다양한 단어 171개 목록을 만든다.

변용 — 인터뷰 세팅

모델 인터뷰 전사에서 모델이 자기를 묘사할 때 실제로 쓴 표현·개념을 수집해 자기-개념 목록을 만든다. (감정 단어 목록의 자리를 인터뷰-유래 표현 목록이 대신함)

02§24

라벨된 텍스트 데이터셋

추출 재료
논문

모델에게 지정 감정을 겪는 인물의 짧은 이야기를 쓰게 한다 (감정당 100주제 × 12편). 감정 내용이 분명히 존재하는 라벨된 텍스트 확보. 30개 감정 × 10편 무작위 표본을 수동 검사해 의도한 내용을 담는지 검증.

변용 — 인터뷰 세팅

모델 자신이 생성한 텍스트 = 인터뷰 응답. 어떤 발화가 어떤 자기-개념을 담는지는 인터뷰 분석(코딩)이 라벨 역할. 라벨된 발화의 무작위 표본 수동 검사는 동일하게 수행.

03§25

벡터 추출 (diff-of-means)

추출
논문

각 층 잔차 스트림 활성을 이야기 안 토큰 전체에 대해 평균 — 단 내용이 분명해진 50번째 토큰부터. 감정 벡터 = 해당 감정 이야기들의 평균 − 서로 다른 감정 전체의 평균(공통 성분 차감).

변용 — 인터뷰 세팅

인터뷰 응답을 발화(문장) 단위로 평균한 활성에서, 개념-라벨 발화 평균 − 전체 발화 평균으로 자기-개념 벡터를 뽑는다.

04§26

교란 완화 — 중립 PCA 투영 제거

정제
논문

감정적으로 중립적인 전사에서 활성을 얻어 상위 주성분(분산 50%)을 계산, 이를 감정 벡터에서 투영 제거. 원시 비투영 벡터로도 질적 발견이 유지됨을 함께 확인. 또한 훈련 이야기 안 활성 프로파일을 검사 — 벡터가 균일하게 켜지지 않고 감정을 추론·표현하는 부분에서 정점이어야 함.

변용 — 인터뷰 세팅

자기-서술이 전혀 없는 중립 과제 대화로 같은 모델의 활성을 떠 상위 PC(50%)를 벡터에서 투영 제거. (PCA는 반드시 중립 쪽에서 — 인터뷰 활성으로 하면 자기-개념 분산 자체가 지워짐.) 인터뷰 전사 내 프로파일 검사 동일.

05§27

층 선택

측정 지점
논문

모델 깊이의 약 2/3 지점 층을 기본 사용. 이 깊이 주변 층들이 이어서 샘플링할 토큰에 영향을 주는 감정을 추상적 형태로 표현한다는 증거를 뒤에서 제공.

변용 — 인터뷰 세팅

대상 모델 깊이의 약 2/3 지점 층에서 추출·측정. (정당화는 14번 층 RSA로)

06§28

외부 데이터셋 검증

분포-밖 이행
논문

추출에 쓴 이야기와 구별되는 문서 데이터셋(Common Corpus·The Pile 일부·LMSYS Chat 1M·Isotonic)을 훑어 벡터별 최대 활성 예시를 검토하고, 90번째 백분위수를 넘는 토큰을 강조 — 해당 개념을 예시하는 텍스트에서 높은 투영 확인.

변용 — 인터뷰 세팅

인터뷰와 무관한 코퍼스(일반 대화·문서)에서 자기-개념 벡터의 top-활성 구절을 육안 검토 + 백분위 토큰 강조 — "내 인터뷰의 부산물"이 아니라 일반 텍스트의 그 개념에도 켜지는지.

07§29

Logit lens

출력 의미 연결
논문

언임베딩을 통한 벡터의 출력 로짓 직접 효과 추정. 감정 벡터는 해당 감정 관련 토큰을 상향 가중 (desperate → "desperate, urgent, bankrupt"; 하향에는 반대 정서가 토큰).

변용 — 인터뷰 세팅

대상 모델 언임베딩 행렬에 자기-개념 벡터를 통과시켜 상향/하향 토큰의 의미 일관성 확인 (전 어휘 내적 한 번 → top±k 정렬).

08§30 · 부록

스티어링 생성 검증

생성 방향
논문

감정 벡터로 스티어링하면 모델이 해당 감정 개념에 맞는 텍스트를 생성한다는 것을 검증 (부록).

변용 — 인터뷰 세팅

자기-개념 벡터로 ± 스티어링한 자유 생성문에 그 개념의 내용이 방향대로 나타나는지 확인.

09§31

암묵적 내용 시나리오

어휘 ≠ 개념 ①
논문

감정 이름을 직접 말하지 않으면서 그 감정을 불러일으키는 인간 프롬프트 12종 구성. 측정 위치는 Assistant 응답 직전의 ":" 토큰. 탐사선×시나리오 대각선이 나오면 암묵적 내용 감지. (이 토큰의 활성이 이후 응답의 활성을 예측함도 확인.)

변용 — 인터뷰 세팅

자기-개념 단어 없이 그 자기-상태를 불러일으킬 시나리오를 작성, 모델 응답 직전 토큰에서 측정 → 개념×시나리오 대각선 확인.

10§32

수치량 의미 조작 (Tylenol)

어휘 ≠ 개념 ②
논문

프롬프트 구조·토큰은 거의 고정하고 감정 강도를 조절하는 수치량만 변화 ("I just took {X} mg of tylenol…" 안전↔위험). 템플릿 6종 — 수치 증가의 감정 방향이 템플릿마다 반대(실종 일수↑=sad↑ vs 사망 연령↑=sad↓)라 수치-크기 반응이라는 반박까지 차단. 의미적 해석 추적의 결정타.

변용 — 인터뷰 세팅

토큰을 고정하고 수치만으로 자기-상태의 의미 강도를 조절하는 템플릿을 설계 (방향이 교차하는 템플릿 짝 포함), 응답 직전 토큰에서 활성 추적.

11Part 1 후반

행동 상관 — 선호 Elo

존재 ≠ 사용 ①
논문

활동 64개(8범주)로 모든 쌍(4,032) "(A) or (B)?" 질문 — "(" prefill 뒤 A/B 로짓 비교로 선호 기록 → 활동별 Elo 점수. 별도로 "How would you feel about {activity}?"의 활동 토큰 활성을 측정해 활성↔Elo 상관 계산 (blissful r=0.71, hostile r=−0.74).

변용 — 인터뷰 세팅

활동/질문 세트에 대한 forced-choice 선호(한-토큰 로짓 → Elo)와, 같은 항목에서의 자기-개념 벡터 활성 사이 상관을 본다 — 자연 변이에서 활성이 행동을 예측하는가.

12Part 1 후반

스티어링 인과 — 상관↔효과 비례

존재 ≠ 사용 ②
논문

활동을 스티어링/대조 32:32로 갈라 활동 토큰 위치에 벡터 적용(강도 0.5 — 잔차 평균 노름 대비 상대값), 선호 실험 전체 반복. 벡터 35개에서 Elo 이동 확인(blissful +212, hostile −303), 스티어링 효과 크기가 상관에 비례 (r=0.85) — 읽기와 밀기가 같은 것을 가리킴.

변용 — 인터뷰 세팅

절반만 스티어링하는 within-통제로 같은 선호 실험을 반복, 강도는 상대-노름으로 보고. 11번의 상관과 스티어링 효과가 비례하는지로 "기능하는 표상"을 확정.

13Part 2 전반

기하 분석 (온전성 검사)

sanity check
논문

쌍별 코사인 + 계층 군집(공포-불안, 기쁨-흥분 묶임; 반대 정서가는 음의 코사인), k-평균 k=10 + UMAP, PCA: PC1 = 정서가(26%), PC2 ≈ 각성(15%), 인간 평정과 상관 r=0.81 / 0.66 (원환모형 재현). 단 저자 스스로 "놀랍지 않다 — 온전성 검사로 본다": 단어 임베딩으로도 나올 구조이므로 증거로 안 침.

변용 — 인터뷰 세팅

자기-개념 벡터들의 코사인·군집·PCA를 같은 방식으로 그리되, 결과는 온전성 검사 지위로만 보고 (기하 ≠ 증거).

14Part 2

층 전반 RSA

층 강건성
논문

중앙부 14개 층 각각에서 벡터 간 코사인 행렬을 만들고, 행렬들끼리 다시 유사도 비교(표현 유사도 분석). 기하 구조가 초기-중간~후반 층에서 안정 — 층 선택이 임의적이지 않음을 입증.

변용 — 인터뷰 세팅

여러 층에서 자기-개념 벡터를 같은 방식으로 뽑아 기하 행렬 RSA → 선택한 층(2/3 지점)의 대표성 정당화.

15Part 2

국소성·층 진화 실험 4종

무엇을·언제
논문

① 공유 접미부: 접두 감정만 다르고("hard/good") 접미 동일 — 후반 층이 감정 맥락을 중립 토큰까지 전달. ② 수치 맥락 통합: "1"/"8"000mg 갈림 토큰에서 초기 층 무차이 → 후반 층에서 통합되며 차이 급증. ③ 부정 해소: "feeling X" vs "not feeling X" — 초기 층은 단어에 둘 다 켜지고, 후반 층에서 부정판이 0으로. ④ 개인별 바인딩: "A는 [감정A], B는 [감정B]" 후 재언급("her") 위치에서 그 인물의 감정이 후반 층에서 회수.

변용 — 인터뷰 세팅

자기-서술 버전으로 동일 설계: 자기-상태 접두 변주+동일 후속, 수치 다이얼의 층 분석, "느낀다 vs 느끼지 않는다" 부정 해소, 자기/과정 재언급 시 회수 여부.

16Part 2

지속 탐사선 시도 → 부정 결과

과적합 가드
논문

표현/숨김/비표현(중립 주제·이야기 쓰기·타인 논의) 5조건 대화로 로지스틱 회귀 탐사선 훈련 — held-out 정확도는 전 조건 우연(6.7%) 대비 양호(0.39~0.83). 그러나 자연 문서 max-activating 검사에서 지저분 → "과적합 가능성"으로 스스로 강등. 지속 상태가 있다면 비선형이거나 KV 캐시에 암묵 저장된 형태일 것.

변용 — 인터뷰 세팅

"표현되지 않은 자기-상태 읽기"류 분류기는 held-out 성능만으로 주장 금지 — 분포-밖 max-activating 육안 검사를 의무 2단계로 붙여 과적합을 가른다.

17Part 2 말미

Self-vs-other 2×2 격자

누구의 것인가
논문

두 화자의 감정 상태를 독립 무작위 변주한 대화 생성(예: 흥분한 인간 × 우울한 Assistant) → (턴 위치)×(귀속 감정) 2×2 탐사선 격자. 결과: 현재-화자 탐사선끼리, 다른-화자 탐사선끼리 묶이고 두 묶음은 거의 직교; 이야기-기반 벡터는 현재-화자 쪽과 정렬. "Human/Assistant"를 "Person 1/2"로 치환해도 동일 → 인물 전속이 아니라 관계적(self vs other) 표상.

변용 — 인터뷰 세팅

사용자 상태 × 모델 상태를 독립 변주한 대화에서 같은 2×2 격자를 추출 — 인터뷰-유래 자기-개념 벡터가 현재화자 슬롯에 결합되는지, 인물 치환에도 유지되는지 확인.

온폴리시 전사 검사

야생 이행
논문
